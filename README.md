# PDF Dropper Red Team Scenarios

This project provides a set of Python scripts and resources to generate PDF droppers for use in Red Team scenarios. Use these tools to simulate common attack vectors, test security controls, and raise awareness about threats involving malicious PDFs.

## Features

- 100% Python implementation
- Customizable payloads for PDF drops
- Designed for penetration testing and Red Team operations
- Easy to use and extend

## Usage

1. **Clone the repository:**
   ```bash
   git clone https://github.com/KTSec234/pdfdropper.git
   cd pdfdropper
   ```

2. **Install dependencies:**
   Make sure you have Python 3.x installed. Then install the required packages:
   ```bash
   pip install -r requirements.txt
   ```

3. **Run the main script:**
   ```bash
   python pdfdropper.py
   ```
   Follow the on-screen instructions to generate and drop your PDF payloads.
